# MindFlow - Gestão Psicológica Inteligente

Plataforma comercial para gestão técnica, financeira e consultiva de pacientes. [cite: 14]

## Identidade Visual [cite: 2]
- **Deep Navy (#1A202C)**: Texto principal e cabeçalhos. [cite: 4]
- **Mind Teal (#2DAAA5)**: Elementos de ação e identidade. [cite: 4]
- **Focus Amber (#F6AD55)**: Alertas e notificações importantes. [cite: 4]
- **Off-White (#F5F7FA)**: Fundo do site para reduzir ofuscamento. [cite: 4]

## Tipografia [cite: 5]
- **Títulos**: Lexend ou Montserrat (facilita a leitura e reduz esforço cognitivo). [cite: 7]
- **Corpo do Texto**: Inter (legibilidade otimizada em telas). [cite: 8]

## Conceito Visual [cite: 9]
O design deve ser minimalista, utilizando cartões (cards) com sombras leves para facilitar a identificação das seções. [cite: 11, 12]
